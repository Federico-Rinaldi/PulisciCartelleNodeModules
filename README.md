# PulisciCartelleNodeModules

 Ho per puro caso ritrovato questa vecchia applicazione, scritta per prova molto tempo fa.
 l'ho ripulita e messa qui su github, giusto per non perdere le informazioni.
 
 Semplicemente pulisce le cartelle node_modules che incontra partendo da una data path passata come parametro.
 Ripeto: è una vecchia prova di studio. 
 Da quel che mi ricordo, su Linux faceva il suo lavoro, su Windows non ne sono cosi sicuro.
 Se qualcuno si vuole divertire ad integrare o riscrivere, può essere un modo per divertirsi un pò.
 
 Ho caricato anche i binaries nel tag release, naturalmente declino ogni responsabilità se cancellate per sbaglio i files del vostro pc.
 
 Buon divertimento!
